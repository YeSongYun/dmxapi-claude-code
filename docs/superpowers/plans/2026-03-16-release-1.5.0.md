## Chunk 2: 提交并发布 Tag

### Task 3: Commit、打 Tag、Push

- [ ] **Step 1: 确认所有改动**

  ```bash

  ```
  预期看到 4 个文件被修改：`dmxapi-claude-code.go`、`install.sh`、`install.ps1`、`install.cmd`

- [ ] **Step 2: 暂存所有修改**

  ```bash
  git add dmxapi-claude-code.go install.sh install.ps1 install.cmd
  ```

- [ ] **Step 3: 提交**

  ```bash
  git commit -m "chore: bump version to 1.5.0"
  ```

- [ ] **Step 4: 打 Tag**

  ```bash
  git tag v1.5.0
  ```

- [ ] **Step 5: Push commit 和 Tag**

  ```bash
  git push && git push origin v1.5.0
  ```
  预期：commit 推送到远端 main 分支，tag v1.5.0 推送成功

- [ ] **Step 6: 验证 Tag 已推送**

  ```bash
  git ls-remote --tags origin | grep v1.5.0
  ```
  预期输出包含 `refs/tags/v1.5.0`
